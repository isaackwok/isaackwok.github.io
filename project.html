              # NoMoney

              NoMoney (opens new window) is a platform for people to earn extra income by helping others in the community, combining LINE Bot for pushing notifications to users. User can search cases by keyword, hashtag and district, hashtags are automatically extracted from the case content using techniques of NLP.

              # Linggle

              An English learning tool based on data mining to big data, developed and maintained by NLPLab (opens new window).

              Linggle (opens new window) is designed for English learners to find out how words and collocations be used frequently in English, with examples from COCA corpus and Google Books. We also provide wildcards for advanced query and membership functionality for saving users' search history.

              Keywords:

              • React.js

              • FirebaseAuth
              • NLP

              # Linggle Booster (Chrome Extension)

              Chrome Store Link (opens new window)

              The shortcut for entering Linggle Booster (opens new window), a website developed and maintained by NLPLab (opens new window).

              LinggleBooster is a corpus reading web system, which allows you to import pdf files or url to get the corresponding explanations and translations to words in the article, which is super helpful to English learners.

              Keywords:

              • Chrome Extension
              • JavaScript

              # LingConc

              LingConc is designed as a self-compiled corpus, as well as a linguistic analysis tool, without the need of internet connection, which is useful for managing and anaylizing papers, news and articles. Users can import articles from either pdf file or url of a news website, then it will extract contents and analyze keywords and linguistic features automatically.

              Paper submitted to poster section of DH Benelux 2020 [Paper 46] (opens new window).

              Keywords:

              • Electron
              • Vue.js
              • PDF.js
              • Node.js
              • Bootstrap
              • NLP

              # GoGoFarming

              GoGoFarming is a Geographic Information System (GIS) for discovering knowledge and realtime information on agriculture and meteorology. We provide information on soil texture, plants distribution, capacity of reservoir, typhoon and weather etc. Besides, users can also subscribe for warning of nearby reservoirs.

              The prize winning work of 2018 International ICT Innovative Services Awards 👑, also the Final Year Project.

              Keywords:

              • jQuery
              • JavaScript
              • HTML/CSS
              • Firebase
              • Open Data
