emitted in a production build, so nothing fake ever goes live.

## Typography

Body text sits on a loose leading with generous spacing. Links look like
[this one](https://isaackwok.com) — hinata gold, underlined softly.

### A quieter subheading

> Blockquotes are indented and set in hai — no left border, spacing does
> the structural work.

A list, for rhythm:

- spacing as hierarchy
- warm ink on kami white
- nothing decorative

## Code

Inline `code` sits on a pale komorebi tint. Blocks scroll sideways when
lines run long:

```js
export function greet(name) {
    return `Hello, ${name} — this line is deliberately long enough to overflow the column.`;
}
```

## 中文

中文段落需要更寬鬆的行距與字距。這一段用來確認 CJK 字形在文章版面中的呼吸感，以及閱讀時間估算能正確計算中文字數。
